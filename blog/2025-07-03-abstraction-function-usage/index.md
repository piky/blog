**Abstraction** แยก _การใช้งาน_ (usage) ออกจาก _การทำงาน_ (function) ของระบบ(ที่ซับซ้อน)  
:::tip เน้นผลลัพธ์
ถ้าเป้าหมาย _เพื่ออำนวยความสะดวก_
:::

:::danger จงเรียนรู้และใส่ใจหลักการทำงาน
ถ้าใช้เป็นเครื่องมือทำมาหากิน
:::
